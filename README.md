# UavRouter BACKEND:

this is the backend/ server for this routing project, it takes a route of waypoints from the frontend and edits them based on risk along the route to path a safer route if needed.

# TO USE:
- must have .NET 10 installed 
- must unzip the DB at the root of the project, you will find the zipped db in the route of the src code uavrouter.db.zip
- must be connected to internet 

# TO RUN:
- dotnet run from terminal in this project 
- run front end (see front end README file)
